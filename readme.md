## About
Test project


## Requirements
1. [PHP] >= 7.2 support.
2. [MySQL] database.
3. [Apache 2.4]


## Used technologies
2. [JQuery]
3. [Twitter Bootstrap]


## Installation

-Clone project
`git clone https://github.com/Leorne/test-art-craft.git`

- Install components
```shell
composer install
```
- Create Database and change DB settings in /configs/db.php

- Migrate database

`composer migrate`
